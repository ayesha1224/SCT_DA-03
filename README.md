# SCT_DA-03
# Task 03 – Interactive Visualization

## Project Overview

This project focuses on building an interactive dashboard using Microsoft Power BI to analyze the HR Employee Attrition dataset.

## Objective

The goal is to understand why employees leave the organization and identify factors that influence employee attrition.

## Tools Used

* Microsoft Power BI
* Data Visualization Techniques

## Dashboard Insights

The dashboard includes the following visualizations:

* Employee Count
* Attrition by Department
* Attrition by Gender
* Attrition by Age
* Job Satisfaction vs Attrition
* Interactive Filters

## Dataset

IBM HR Employee Attrition Dataset

## Outcome

The dashboard helps visualize patterns in employee attrition and allows users to explore factors affecting employee turnover through interactive filters.
